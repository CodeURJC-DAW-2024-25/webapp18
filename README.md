## Functionalities

### Entities

Apartments, users, review, reservation.

| Activities | Registered client | Unregistered client | Administrator |

| Search / filter apartments | ✔      | ✔       | ✔       |
| Write reviews       | ✔  | ❌   | ❌  |
| Modify the apartments DB       | ❌  | ❌   | ✔  |

| Make reservations      |  ✔ | ❌   | ✔  |
| Modify reservations       | ✔  | ❌  | ✔  |
| View reservations history       | ✔  | ❌  | ✔  |
